# My-Denim

## Description
"My-Denim" est un projet qui inclut un jeu "Minesweeper" jouable dans la console, développé en C++.

## Prérequis
Avant de compiler le projet, assurez-vous d'avoir installé les éléments suivants :

- **CMake** (version 3.10 ou supérieure)
- **Make** (pour les systèmes basés sur Unix)

## Compilation

### Avec CMake

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Jaures229/My-Denim.git
   cd My-Denim
   ```

2. Créez un répertoire de construction :
   ```bash
   mkdir build
   cd build
   ```

3. Exécutez CMake pour configurer le projet :
   ```bash
   cmake ..
   ```

4. Compilez le projet :
   ```bash
   make
   ```

### Avec Makefile

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Jaures229/My-Denim.git
   cd My-Denim
   ```

2. Compilez le projet directement avec Make :
   ```bash
   make
   ```

## Lancer le jeu
Après la compilation, vous pouvez lancer le jeu avec la commande suivante :
```bash
./MyDemin
```
